# Football Match Scedule
Android Football Match Schedule using MVP Pattern

The app uses the following libraries / topics:
- MVP Pattern
- Anko Design
- Anko Commons
- Anko SQLite
- Picasso

API:
Football Match Schedule data from https://www.thesportsdb.com/
